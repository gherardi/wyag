# wyag

A lightweight version control system implementation in Python.

## Description

`wyag` is an educational project that demonstrates the core concepts of Git by implementing a simplified version control system from scratch. It covers repository initialization, object storage, indexing, and commit history management.

## Usage

```bash
./wyag init .
./wyag add <file>
./wyag commit -m "message"
```

## Getting Started

1. Clone the repository
2. Install Python 3.10+
3. Run `chmod +x wyag` to make the file executable
